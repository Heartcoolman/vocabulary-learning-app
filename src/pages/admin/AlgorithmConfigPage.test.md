# 算法配置页面测试指南

## 测试目标
验证算法配置页面的所有功能是否正常工作。

## 前置条件
1. 以管理员身份登录系统
2. 访问 `/admin/algorithm-config` 路由

## 测试用例

### 1. 页面加载测试
- [ ] 页面能够正常加载
- [ ] 显示所有配置区块（6个）
- [ ] 显示当前配置值
- [ ] 显示默认配置值对比

### 2. 遗忘曲线参数测试
- [ ] 能够查看当前复习间隔序列
- [ ] 能够添加新的复习间隔
- [ ] 能够删除复习间隔（至少保留1个）
- [ ] 能够修改复习间隔的值
- [ ] 显示默认值对比

### 3. 难度调整参数测试
- [ ] 连续答对阈值滑块工作正常（范围3-10）
- [ ] 连续答错阈值滑块工作正常（范围2-5）
- [ ] 调整间隔输入框工作正常
- [ ] 实时显示当前值
- [ ] 显示默认值对比

### 4. 优先级权重测试
- [ ] 4个权重滑块工作正常
- [ ] 实时计算权重总和
- [ ] 总和不等于100%时显示错误提示
- [ ] 总和等于100%时显示绿色确认
- [ ] 显示默认值对比

### 5. 掌握程度阈值测试
- [ ] 表格显示5个级别的阈值
- [ ] 能够修改连续答对次数
- [ ] 能够修改最低正确率（0-1）
- [ ] 能够修改最低得分（0-100）
- [ ] 正确率显示百分比转换

### 6. 单词得分权重测试
- [ ] 4个权重滑块工作正常
- [ ] 实时计算权重总和
- [ ] 总和不等于100%时显示错误提示
- [ ] 总和等于100%时显示绿色确认
- [ ] 显示默认值对比

### 7. 答题速度评分标准测试
- [ ] 能够修改优秀阈值
- [ ] 能够修改良好阈值
- [ ] 能够修改一般阈值
- [ ] 能够修改较慢阈值
- [ ] 显示毫秒和秒的转换
- [ ] 显示默认值对比

### 8. 配置验证测试
- [ ] 权重总和不为100%时无法保存
- [ ] 复习间隔非递增时显示错误
- [ ] 阈值超出范围时显示错误
- [ ] 所有验证错误都显示在页面顶部

### 9. 保存功能测试
- [ ] 配置有效时保存按钮可用
- [ ] 配置无效时保存按钮禁用
- [ ] 保存成功后显示成功提示
- [ ] 保存失败后显示错误信息

### 10. 重置功能测试
- [ ] 点击"恢复默认值"按钮显示确认对话框
- [ ] 对话框显示警告信息
- [ ] 点击"取消"关闭对话框，不重置
- [ ] 点击"确认重置"恢复所有默认值
- [ ] 重置后显示成功提示

### 11. UI/UX测试
- [ ] 所有区块使用卡片样式
- [ ] 修改过的配置显示"已修改"标记
- [ ] 滑块拖动流畅
- [ ] 输入框聚焦时有蓝色边框
- [ ] 按钮有悬停效果
- [ ] 页面响应式布局正常

## 预期结果
所有测试用例都应该通过，页面功能完整且用户体验良好。

## 注意事项
1. 配置修改会影响所有用户的学习体验，请谨慎操作
2. 建议在测试环境中进行充分测试后再应用到生产环境
3. 重置操作不可撤销，请确认后再执行
