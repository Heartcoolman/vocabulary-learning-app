# Getting Started

欢迎使用词汇学习应用组件库！本文档将帮助你快速了解如何使用这些组件。

## 项目概述

这是一个基于 React + TypeScript + Tailwind CSS 的词汇学习应用组件库。

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Phosphor Icons
- Vite

## 安装

```bash
pnpm install
pnpm run storybook
```

## 核心组件

### 基础组件

- Button - 按钮组件
- Input - 输入框组件
- Card - 卡片组件
- Modal - 模态框组件
- Toast - 消息提示组件

### 业务组件

- WordCard - 单词学习卡片
- MasteryProgress - 掌握进度展示
- Navigation - 导航栏组件
- FileUpload - 文件上传组件
- LineChart - 折线图组件

## 使用示例

### 按钮组件

```tsx
<button className="btn-primary">主要按钮</button>
<button className="btn-secondary">次要按钮</button>
```

### 输入框

```tsx
<input className="input-enhanced" placeholder="请输入..." />
```
