# Danci Android E2E 测试用例

## 1. 应用启动测试

- [ ] 冷启动时间 < 2秒
- [ ] 首页正确加载
- [ ] WebView 正常渲染

## 2. 离线学习流程

- [ ] 无网络状态下可以学习
- [ ] 学习状态正确保存到本地 SQLite
- [ ] 答题记录正确记录

## 3. TTS 发音测试

- [ ] 单词发音正常播放
- [ ] 不同语速设置生效
- [ ] 中英文发音切换

## 4. 视觉疲劳检测测试

- [ ] 摄像头权限请求
- [ ] 权限拒绝时的降级处理
- [ ] Web MediaPipe 正常工作

## 5. 数据同步测试

- [ ] 网络恢复后自动同步
- [ ] 冲突解决策略正确
- [ ] 同步状态 UI 显示

## 6. 性能测试

- [ ] 内存占用 < 150MB
- [ ] 列表滚动流畅 (60fps)
- [ ] 页面切换无卡顿

## 7. 兼容性测试

- [ ] Android 7.0 (API 24)
- [ ] Android 10 (API 29)
- [ ] Android 13 (API 33)
