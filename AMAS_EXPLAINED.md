# AMAS系统通俗解读 - 一个"懂你"的背单词AI

## 一句话总结

**AMAS就像一个懂你的学习管家**，它能感知你累不累、专不专心、学得开不开心，然后自动调整你的学习内容，在最合适的时间让你复习最需要复习的单词。

---

## 1. AMAS是什么？

AMAS全称是"自适应多维度用户感知智能学习算法引擎"，名字很长，但其实做的事情很简单：

> **它就是一个"读心术"系统，能猜出你现在的学习状态，然后给你安排最合适的学习内容。**

想象一下，你有一个超级贴心的私人老师：
- 当你注意力不集中时，他会让你学简单点的单词
- 当你累了，他会提醒你休息
- 当你学得很嗨，他会给你一些挑战性的内容
- 他还能准确预测哪些单词你快要忘了

这就是AMAS干的事！

---

## 2. AMAS怎么"读心"？它看哪些指标？

AMAS会实时追踪你的4个状态，我们叫它们 **A、F、C、M**：

| 指标 | 全称 | 说明 | 怎么检测的？ |
|------|------|------|-------------|
| **A** | Attention（注意力） | 你有没有走神？ | 看你答题速度是否变慢、是否频繁停顿 |
| **F** | Fatigue（疲劳度） | 你累不累？ | 看你错误率是否上升、反应是否变慢 |
| **C** | Cognitive（认知能力） | 你记得住吗？ | 看你的正确率、反应时间稳定性 |
| **M** | Motivation（动机/情绪） | 你想学吗？ | 看你答对时是否开心、答错时是否想放弃 |

**举个例子**：

假设你学了30分钟单词，AMAS发现：
- 你答题速度越来越慢（A下降）
- 错误率开始上升（F上升）
- 正确率还行（C正常）
- 连续答错后你开始随便点（M下降）

AMAS就会判断："哎，这个用户开始累了，注意力也不行了，应该让他休息或者换点简单的内容。"

---

## 3. AMAS怎么预测你会忘记单词？（ACT-R记忆模型）

这是AMAS最厉害的部分！它用了一个叫**ACT-R**的认知科学模型，这是心理学家研究了几十年的成果。

### 核心思想：遗忘曲线

你背了一个单词，如果不复习，记忆会随时间衰减：

```
记忆强度
100% │▓▓▓▓▓▓▓
     │  ▓▓▓▓▓
 50% │    ▓▓▓▓
     │      ▓▓▓
     │        ▓▓
  0% └────────────→ 时间
     刚学  1天  3天  7天
```

但如果你复习了，记忆会被"强化"，而且下次忘得更慢：

```
记忆强度
100% │▓▓▓▓▓▓▓         第2次复习后
     │  ▓▓▓▓▓  ▓▓▓▓▓▓▓▓▓▓▓▓
     │    ▓▓▓▓     ▓▓▓▓▓▓▓
 50% │      ▓▓▓        ▓▓▓▓▓
     │        ↑复习         ▓▓
  0% └──────────────────────→ 时间
```

### AMAS怎么用这个？

AMAS会记录你每个单词的复习历史，然后用数学公式计算：

1. **这个单词你现在能想起来的概率是多少？**（比如85%）
2. **如果不复习，多久后你会忘到只剩70%？**（比如3天后）
3. **所以最佳复习时间是什么时候？**（3天后！）

**简单说就是**：AMAS能预测你哪些单词快要忘了，然后在你忘之前提醒你复习。

---

## 4. AMAS怎么判断你"学会"了一个单词？

传统背单词软件只看一个指标：你答对了几次。

但AMAS更聪明，它看**三个指标**的综合得分：

| 指标 | 权重 | 说明 |
|------|------|------|
| SRS等级 | 30% | 你在间隔重复系统里的掌握等级（0-5级） |
| ACT-R预测 | 50% | 根据遗忘曲线，你现在能回忆起来的概率 |
| 近期正确率 | 20% | 最近几次答题的正确率 |

**公式**：

```
综合得分 = 0.3 × SRS等级 + 0.5 × 回忆概率 + 0.2 × 近期正确率

如果 综合得分 ≥ 0.7，就认为你"学会"了！
```

**还有一个细节**：如果你很累（疲劳度高），系统会降低置信度。

为什么？因为你累的时候就算答对了，也不能说明你真的记住了，可能只是碰巧想起来的。

---

## 5. AMAS怎么决定给你什么内容？（LinUCB算法）

这是一个**强化学习**算法，简单说就是：

> 系统会不断尝试不同的学习策略，看看哪种策略对你效果最好，然后越来越多地使用好的策略。

### 系统能调整哪些东西？

| 参数 | 说明 | 范围 |
|------|------|------|
| 复习间隔 | 同一个单词多久复习一次 | 短 → 长 |
| 新词比例 | 新单词和复习单词的比例 | 0% → 50% |
| 难度等级 | 选择简单/中等/困难的单词 | easy/mid/hard |
| 提示级别 | 给多少提示信息 | 无 → 完整提示 |

### 决策逻辑

```
如果 你很累（F高）+ 注意力低（A低）:
    → 选择简单内容，多给提示，减少新词

如果 你很嗨（M高）+ 认知能力强（C高）:
    → 选择挑战内容，少给提示，可以多学新词

如果 你快要忘记某些单词（ACT-R预测概率低）:
    → 优先安排这些单词复习
```

## 6. 整个流程是怎样的？

```
你答了一道题
    ↓
记录：答案、用时、是对是错
    ↓
AMAS分析你的状态（A/F/C/M）
    ↓
更新单词记忆轨迹
    ↓
ACT-R计算所有单词的"快忘程度"
    ↓
LinUCB选择下一步策略
    ↓
给你推送最合适的下一道题
    ↓
（循环）
```

## 9. 总结

AMAS系统就像一个**超级智能的学习管家**：

1. **它能感知你** - 知道你累不累、专不专心
2. **它能预测遗忘** - 知道你哪些单词快忘了
3. **它能智能决策** - 在最合适的时机让你学最合适的内容
4. **它会不断学习** - 越用越懂你，越来越准

最终目标只有一个：**让你用最少的时间，记住最多的单词，而且不容易忘！**
