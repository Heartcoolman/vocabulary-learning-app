# 更新日志

## v1.6.5

### 2026-01-30 - UMM记忆模型、VARK学习风格与AMAS持久化增强

**UMM 通用记忆模型**

- 引入 Universal Memory Model (UMM)，替换原有记忆算法
- MDM 记忆动力学模型 - 替代原 ACT-R 遗忘曲线
- MSMT 多尺度记忆痕迹 - 多时间尺度记忆强度追踪
- 支持个性化记忆曲线拟合

**决策层算法替换**

- IGE 信息增益探索 - 替代 Thompson Sampling，基于信息论的词汇选择
- SWD 相似度加权决策 - 替代 LinUCB，利用词汇语义相似度优化决策

**VARK 学习风格评估**

- 新增 VARK 学习风格分类器（视觉/听觉/读写/动觉）
- 基于交互行为自动推断学习偏好
- 为内容呈现提供个性化适配依据

**AMAS 运行时状态持久化**

- 学习状态跨会话保存，确保算法连续性
- 新增 amas_runtime_states 数据表
- 支持多设备状态同步

**管理员工作流监控**

- 新增工作流监控面板，实时追踪系统任务
- 在线用户详情展示
- 系统广播功能

**Bug 修复**

- 修复用户流订阅权限验证漏洞
- 修复测试用例兼容性问题
