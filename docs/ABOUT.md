# 关于我们

单词学习助手是一款基于 AMAS（Adaptive Multi-modal Assessment System）自适应学习系统的智能单词记忆应用。

## 我们的使命

通过先进的认知科学和机器学习技术，为每位学习者提供个性化的学习体验，让单词记忆变得更加高效、有趣。

## 核心技术

- **自适应学习算法**：基于遗忘曲线和间隔重复原理，智能调整复习计划
- **多维度评估**：综合考虑记忆强度、学习状态、认知负荷等因素
- **实时反馈**：即时分析学习表现，动态优化学习路径
