# 隐私政策

**最后更新日期：2025年1月**

我们非常重视您的隐私。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。

## 1. 信息收集

我们可能收集以下类型的信息：

### 账户信息

- 注册时提供的邮箱地址、用户名等

### 学习数据

- 您的学习进度、答题记录、学习时长等

### 设备信息

- 浏览器类型、操作系统、IP地址等

## 2. 信息使用

我们使用收集的信息用于：

- 提供和改进我们的服务
- 个性化您的学习体验
- 分析和优化学习算法
- 发送服务相关通知

## 3. 信息保护

我们采取适当的技术和组织措施来保护您的个人信息，包括：

- 数据加密传输（HTTPS）
- 安全的数据存储
- 访问控制和权限管理

## 4. 信息共享

我们不会出售、交易或以其他方式向外部方转让您的个人信息，除非：

- 获得您的明确同意
- 法律要求或政府机关依法要求
- 保护我们的权利、财产或安全

## 5. 您的权利

您有权：

- 访问和更正您的个人信息
- 删除您的账户和相关数据
- 撤回同意（不影响撤回前的处理合法性）

## 6. Cookie 使用

我们使用 Cookie 和类似技术来：

- 保持您的登录状态
- 记住您的偏好设置
- 分析服务使用情况

## 7. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并注明更新日期。

## 8. 联系我们

如对本隐私政策有任何疑问，请通过 GitHub Issues 联系我们。
